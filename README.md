# Zamanlay-c-
Glass Timer, modern buzlu cam (Glassmorphism) tasarımıyla estetiği ve işlevselliği birleştiren, kullanıcı dostu bir odaklanma asistanıdır. Masaüstünüzde yer kaplamayan şık arayüzü ve sistem tepsisi (tray) desteğiyle, verimliliğinizi artırmak için tasarlanmış minimalist bir zamanlayıcıdır.
